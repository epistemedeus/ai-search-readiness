# The State of AI-Search Readiness 2026

An open dataset + study: how readable are 189 well-known companies to AI search engines (ChatGPT, Perplexity, Google AI), scored across 10 industries.

**Live study:** https://epistemedeus.github.io/ai-search-readiness/
**Full data (CSV, CC-BY):** https://github.com/epistemedeus/ai-readiness/blob/main/data/ai-search-readiness-2026.csv
**Interactive leaderboard:** https://samedaydesk.com/reports/ai-search-readiness-leaderboard-2026.html
**Check your own site (free):** https://samedaydesk.com/tools/ai-readiness — or `npx github:epistemedeus/ai-readiness yoursite.com`

Healthtech (avg 63) and news media (64) score worst; marketing agencies (92) best. OpenAI and GitHub scored a D; Klarna an F.
